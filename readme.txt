== Description ==
Everything is customizable. if you need more style you can customize css.
You will get Promotional slider, 2 types of Category slider.

FEATURES LIST
	1. Unlimited Category slider with variation
	3. Promotional slider
	4. Add custom thumbnail.
	5. Compatible with any Theme
	6. Display specific categories in the slider.
	7. Swiper, powerful, and fast loading.
	8. Responsive and mobile-ready.
